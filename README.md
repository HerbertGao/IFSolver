# IFSolver

A tool to find passcodes of IFS @ Home.

> This version of IFSolver is currently WIP.

## Usage

- Put `Portal_Export.csv` into `./input`.
- Put `ifs.jpg` into `./input`. (Optional)
- Run `python3 main.py`. (With optional args)
